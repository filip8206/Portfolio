First JavaScript project

May 2023

The challenge

In this project, we got familiar with JavaScript. The goal was to create a minigame using only HTML, CSS and JavaScript. I chose to make the classic Nintendo game 'Loves me...?'. A strategic 2-player game where you have to make sure you win, or the other player loses.

The Focus

The main focus of this project was to learn how JavaScript works and to create a small game in only a few weeks. Not putting too much time in the styling of the website.

Try it out

Try the game out for yourself!

Game rules

In this game you have a flower with a random number of leaves. Every turn, you can pick 1, 2 or 3 petals from the flower. Every time a petal is picked the flower changes to Loves me or Loves me not. When the flower ends in Really loves me, the player who picked the last petal gets a point. When the flower ends in Really loves me not, the other player gets a point. So think carefully!
