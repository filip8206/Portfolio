PinqPonq Studio

December 2022

The challenge

The goal of this project was to create an intuitive user interface to create your own PinqPonq messenger bag. The main challenge was to fit the loads of customization options into an easy to use interface for desktop. Focusing more on the layout than on the style of the website.

A clear layout

The content of the page is organized between three clear columns, making the different customization-options very clear for the user. This also ensures that the user can explore all the customization-options in a natural and linear order.
