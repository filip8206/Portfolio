PinqPonq Studio

December 2022

The challenge

The goal of this project was to create an intuitive user interface to create your own PinqPonq messenger bag. The main challenge was to fit the loads of customization options into an easy to use interface for desktop. Focusing more on the layout than on the style of the website.

A clear layout

The content of the page is organized between three clear columns, making the different customization-options very clear for the user. This also ensures that the user can explore all the customization-options in a natural and linear order.

Explore the design

Go through the design yourself on the website of Adobe XD!

Visit Adobe XD

The result

This project put my skills to the test to create an intuitive design with lots of functionality. Because of the lots of customization-options I had to include in my design, I had to experiment with different lay-outs and use the available screen estate as efficiently as possible, without making the design too cluttered. In the end, I'm very happy with how the design turned out.
